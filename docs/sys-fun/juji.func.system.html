ask-question

(ask-question rep qn)(ask-question rep qn wording)(ask-question rep qn wording mode)

Ask open-ended question. If wording is given, use the wording; otherwise use the wording or heading from the question template. Mode keyword can be :first-time or :repeating, for :first-time, we use wording if available; for :repeating, we use heading.

get-followup-question

deprecated

(get-followup-question rep followed)(get-followup-question rep followed cond-key-or-html-vec)

Given a question and a condition keyword, return the corresponding followup question. If the condition keyword is not given, return the followup question under the :default key. If conditional followup does not exit, we double check default followup. NOTE: currently only conditional single choice followup uses html-vec. Need to expand the let condition if new html-vec question is introduced.
